Анализ лояльности пользователей Яндекс Афиши. Цель проекта - предоставить заказчикам обоснованные рекомендации и данные, которые помогут им принять взвешенные решения не только по привлечению новых пользователей, но и удержанию, превращению их в постоянных. 
В ходе выполнения проекта решим следующие задачи: познакомимся с данными, проверим их корректность и проведем предобработку, проведем исследовательский анализ данных, сопровождающийся необходимыми визуализациями.
